# system-design-playground

A playground for implementing and experimenting with various system design components.

## Project Structure

- `src/` - Main entry points and examples
- `designs/` - Individual system design implementations
  - `rate-limiter/` - Rate limiter implementation

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

Run the project in development mode (with hot reload):

```bash
npm run dev
```

### Build

Compile TypeScript to JavaScript:

```bash
npm run build
```

### Run

Run the compiled JavaScript:

```bash
npm start
```

### Watch Mode

Watch for file changes and automatically recompile:

```bash
npm run watch
```

### Clean

Remove compiled output:

```bash
npm run clean
```

## Rate Limiter

The rate limiter implementation is located in `designs/rate-limiter/src/`.
